# Personal Website

Minimal holding page for [gaoshanghui.com](https://www.gaoshanghui.com/).

## Stack

- Next.js
- React
- TypeScript
- App Router
- Tailwind CSS

## Scripts

```bash
pnpm dev
```

Starts the local development server at `http://localhost:3000`.

```bash
pnpm build
```

Builds the app for production.

```bash
pnpm start
```

Runs the production build locally.

```bash
pnpm lint
```

Runs ESLint.
